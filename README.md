# Typescript Exercises

The goal: Let everyone play with many different TypeScript features and get an overview of TypeScript capabilities and principles.

## Things to cover

1. Basic typing.
2. Refining types.
3. Union types.
4. Merged types.
5. Generics.
6. Type declarations.
7. Module augmentation.
8. Advanced type mapping.
